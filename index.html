Fresh Glass: an iOS Application

Fresh Glass is an iOS Application implemented with Kinoma JS and written in JavaScript. It works as a remote control for a simulated connected window device capable of changing its tint and display image. It was developed as a class project for CS 160: User Interfaces in Spring 2015.
Hackster Project Page

Elo Calcuator

The Elo System is a method of ranking players originally developed for Chess. This Elo Calculator takes in an input file of an player adjacency matrix showing wins/losses in no particular order, randomizes the order of the games 1000 times and uses those random orderings to calculate the ELO of each player. It then outputs a file showing the calculated ELO of each player. This was developed as a personal project.
Github Repo

Stateless Firewall

This stateless firewall was designed to run at end-hosts and filter traffic on a packet-by-packet basis based on basic firewall rules such as IP, Port, Protocol and DNS query. It was developed and tested with the help of a Linux-based virtual machine.
Github Repo

Sun-activated Alarm Clock

The sun-activated alarm clock is a mechanical device built with repurposed parts from a broken inkjet printer. When ambient sunlight (like the rising sun) hits a photocell indicator, the alarm goes off. A waving person moves up and down, a fan moves from side-to-side on the old printer carriage, and a small bell goes off. It was developed as a class project for EE40: Microelectronic Circuits.
